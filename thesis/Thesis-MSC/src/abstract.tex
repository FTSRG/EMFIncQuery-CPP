\chapter*{Kivonat}\addcontentsline{toc}{chapter}{Kivonat}

Napjainkban a \emph{Modell Vezérelt Szoftverfejlesztés} egyre gyakoribb
eszközévé válik a szoftverfejlesztésnek, ami azt is jelenti hogy az igény
támogató eszközökre magasabb mint valaha. A \emph{Modell Vezérelt
Szoftverfejlesztési} paradigma egyik ágazata a \emph{Futás Idejû Modellezés},
melynek alapja az absztrakt rendszermodellek futás idejû használata információ
tárolására. Futás idejû modellek használata során létfontosságú, hogy képesek
legyünk a modell feletti lekérdezések futtatására hatékony módon. A diplomaterv
célja hogy erre a problémára adjon egy lehetséges megoldást. Ezen probléma
megoldására már létezik több eszköz és módszertan, de az eszközök nagy része
magasabb szintû, gyakran kevésbé hatékony nyelveket alkalmaz mint a \emph{Java}.
A célom egy bõvítmény létrehozása \emph{\EIQ{}} fölé, mely egy létezõ, Java
alapú eszköz. Ezen bõvítmény feladata hogy lehetõvé tegye \CPP{} kód
generálását, melynek segítségével lehetséges lekérdezések futtatása futás idejû
modellek felett \emph{Lokális Keresési} módszer segítségével.

\chapter*{Abstract}\addcontentsline{toc}{chapter}{Abstract}

Nowadays \emph{Model Driven Engineering} becomes more and more common in the
field of software engineering, which means the demand for supporting tools is higher than
ever. One branch of the \emph{Model Driven Engineering} paradigm is
\emph{Runtime Modeling}, which focuses on using the abstract system models
during runtime to hold valuable information. When using runtime models, it is
paramount to be able to run queries over these models in an efficient manner.
The aim of this thesis is to give a possible solution for this problem. There
are several possible solutions for this problem with different tools, but most
of them are written in higher level, most of the time less efficient languages
like \emph{Java}. My goal is to create an extension over \emph{\EIQ{}}, an
already existing Java based tool, which allows the generation of \CPP{} code to
execute queries over runtime models using an approach called \emph{Local
Search}.
